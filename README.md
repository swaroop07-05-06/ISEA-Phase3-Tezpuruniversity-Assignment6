# ISEA-Phase3-Tezpuruniversity-Assignment6
# GUI Based Multi Client Chat Application

## Objective

Develop a GUI-based multi-client chat application using Python sockets and Tkinter by reusing the Assignment 5 server.

---

## Software Requirements

- Ubuntu 22.04
- Python 3
- Tkinter
- Socket Programming
- Threading
- CSV
- VirtualBox
- Mininet
- Wireshark

---

## Network Topology

Server
|
|------ Client 1
|
|------ Client 2
|
|------ Client 3

TCP Port: 5000

---

## Execution Steps

### Start Server

python3 server.py

### Start Client

python3 client_gui.py

Enter:

- Server IP
- Username

Click Connect.

---

## Implementation

Features:

- Multi-client communication
- GUI using Tkinter
- Broadcast messaging
- Private messaging
- Online users panel
- Chat history
- Disconnect support
- Join/Leave notifications
